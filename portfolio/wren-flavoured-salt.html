Wren flavoured salt

Swinburne (University Project)
2021
Packaging Design Illustration

About

The goal of this project was to create a set of 3 packages for a boutique product using only 2 Pantone spot colours. For Wren flavoured salts, I was inspired by native Australian ingredients and illustrated each ingredient for the packaging. Utilising overprinting, each package has a variety of shades using only 2 spot colours. I chose a green to be used consistently across the brand with each sub-product having a second unique colour to match the ingredient.
